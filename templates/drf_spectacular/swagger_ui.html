Как пользоваться

Краткая инструкция (без «айти»-сленга):
  1. Войти в систему API. В блоке «Быстрый вход» ниже введите логин и пароль — авторизация включится автоматически (ничего копировать не нужно).
  2. Найти нужный раздел. Ниже список сгруппирован по темам: авторы, книги, выдачи и т.д. Строка Filter by tag — это поиск по названию операции.
  3. Выполнить запрос. Раскройте нужную строку → кнопка «Try it out» → заполните поля → «Execute». Ответ появится ниже.

Подписи к операциям и разделам — на русском. Кнопки Swagger (Try it out, Execute) остаются на английском — смысл тот же, что в инструкции выше.

Быстрый вход

Быстрый вход (чтобы ничего не копировать):

Введите логин и пароль — мы получим токен и автоматически включим авторизацию для всех запросов ниже.

После входа вы можете выполнить любой запрос: откройте строку → Try it out → Execute.
